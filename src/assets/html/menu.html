Hotkeys:

- F2: Screenshot and copy to clipboard

- F4: Return to https://kirka.io

- F5: Reload page

- F6: Load URL

- F7: Copy URL

- F11: Fullscreen

- F12 or Ctrl + Shift + I: Open Devtools

- Ctrl + C: Copy image preview to clipboard

- Shift + Click on a name in global chat: Open player profile

- Shift + Click on a link in friends list: Copy game link
